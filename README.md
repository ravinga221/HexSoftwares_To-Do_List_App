## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/ravinga221/To-Do_List_App.git
   ```
2. Navigate to the project folder:
   ```bash
   cd To-Do_List_App
   ```
3. Open `index.html` in your web browser.

### Usage
1. Type your task in the input field
2. Press "Add" or hit Enter to add it to your list
3. Click the checkbox next to a task to mark it as completed
4. Click the delete button (typically an "X" or trash icon) to remove tasks
